# Security Policy

Vielen Dank, dass Sie zur Sicherheit von MPV beitragen.

## Supported Versions

Aktuell gilt: Sicherheitsupdates werden für die jeweils neueste Version im Hauptbranch bereitgestellt.

| Version | Supported |
| --- | --- |
| main (aktuell) | :white_check_mark: |
| ältere Stände | :x: |

## Reporting a Vulnerability

Bitte melden Sie vermutete Sicherheitslücken **nicht öffentlich** über ein GitHub-Issue.

Stattdessen:
1. Kontaktaufnahme mit einer privaten Beschreibung des Problems.
2. Wenn möglich: Reproduktionsschritte, betroffene Version, potenzieller Impact.
3. Optional: Vorschlag für Mitigation/Fix.

Bis ein dedizierter Security-Kontakt hinterlegt ist, bitte das Maintainer-Team direkt und vertraulich kontaktieren.

## Disclosure Process

- Eingang wird zeitnah bestätigt.
- Wir prüfen Reproduzierbarkeit und Schweregrad.
- Bei bestätigter Lücke erfolgt ein Fix und anschließend koordinierte Veröffentlichung.
